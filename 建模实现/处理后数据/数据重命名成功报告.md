## 🎯 **新的命名规范**

### **标准格式**：
```
{采样频率}_{传感器位置}_data_{故障类型}_{故障尺寸}_{载荷}_{片段编号}
```

### **命名组成说明**：
1. **采样频率**: `12kHz` / `48kHz`
2. **传感器位置**: `DE`(驱动端) / `FE`(风扇端) / `Normal`(正常)
3. **固定字符**: `data`
4. **故障类型**: `B`(滚动体) / `IR`(内圈) / `OR`(外圈) / `N`(正常)
5. **故障尺寸**: `0007` / `0014` / `0021` / `0028`（英寸，仅故障数据有）
6. **载荷**: `0` / `1` / `2` / `3`（马力）
7. **片段编号**: 从原始数据分割的片段序号
